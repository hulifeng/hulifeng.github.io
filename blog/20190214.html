# 2019-02-14

# 1. 主页图片显示

  • 在 .vuepress 文件夹中创建 public/img 文件夹
  • 将图片上传至当前目录中
  • 然后在根目录 README.md 中调用：heroImage: /img/logo.png

# 2. 若是文件夹中包含多文件，配置 sidebar 一定要用多个侧边栏

sidebar: {
    '/blog/': [
        '',
        '20190214'
    ]
}
